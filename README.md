# 3D Robot Builder 

An interactive 3D sandbox mini-app where users can assemble a robot from various parts, change their colors, view dynamic stats, and trigger simple actions. Developed as a technical test task.

## ⚙️ Features
* Mix and match parts (Head, Body, Legs) using UI controls.
* Real-time recalculation of total **Weight** and **Power**.
* Color customization for each equipped part.
* 360-degree interactive robot rotation.
* "Action" button triggers a procedural jump animation.


* **Time spent:** ~2.5 hours.
